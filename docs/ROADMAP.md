# Roadmap

## 0.1 Initial state

- working exhibition sequence
- four facet previews
- direct index
- content collections
- initial project and entry records
- static Docker deployment

## 0.2 Art direction pass

- replace system-font prototype with final licensed typography
- commission or create original visual material
- refine transitions per facet
- tune mobile compositions independently
- design an exhibition loading and state-restoration model

## 0.3 Publishing surface

- RSS
- Open Graph images
- tag pages
- search
- image pipeline and captions
- optional newsletter or email subscription

## 0.4 Living exhibition

- session-aware exhibition state
- rotating current exhibition
- cross-links between works by concept rather than only category
- current signals sourced from content

## Deliberately excluded from initial ship

- accounts
- comments
- public follower counts
- likes
- heavy CMS
- authentication
- analytics that identify individual visitors
