# Design system

## Direction

Warm editorial materiality with the compositional confidence of an art exhibition.

The site should not resemble a gallery brochure. The interface itself is the exhibition.

## Materials

- paper: warm, clean, never fake parchment
- ink: deep neutral black
- wine: rare emphasis, annotations, and active marks
- petrol: synthesis and deeper system surfaces
- night: audio and infrastructure contrast

## Typography

Three voices:

- serif for identity, work titles, and prose
- sans-serif for practical interface copy
- monospace for status, dates, routes, and system evidence

No font files are committed in the initial state. The stack uses high-quality system fallbacks so the repo works immediately. A licensed or self-hosted typeface can be selected later.

## Composition

- one dominant idea per exhibition frame
- layered evidence rather than empty minimalism
- large shifts in scale
- no default card grid on the landing
- deeper pages may use conventional lists and indexes
- visible restraint must imply selection, not lack of material

## Motion

Motion changes the composition instead of decorating it.

- frame switching rather than long scroll choreography
- reduced-motion support is mandatory
- no cursor trails, particles, or cinematic loaders
- direct routes must never depend on animation
